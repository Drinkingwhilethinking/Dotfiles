# 🌿 Hyprland Dotfiles

## 🧠 About
My personal Hyprland setup focused on a clean and minimal aesthetic.

## ⚙️ Setup
- WM: Hyprland
- Bar: Waybar
- Terminal: Kitty
- Launcher: Rofi
- File Manager: Yazi

## 🎨 Features
- Minimal layout
- Blur + transparency
- Clean Rofi launcher
- Custom Waybar

## 📁 Structure
- hypr → window manager config
- waybar → top bar
- kitty → terminal config
- rofi → launcher config
- yazi → file manager config
- HY_WL → wallpapers

## ⚠️ Notes
These are my personal configs. You may need to tweak them for your system
